# FRAMEWORK.md - Ponte legada

Este arquivo existe para compatibilidade com leituras antigas do repo Mi6.

O protocolo LOGUS vigente vive em:

- `LOGUS.md` - especificacao algoritmica do framework;
- `dialogos/` - principios operacionais materializados;
- `CLAUDE.md` - regras de operacao para Claude Code;
- `.claude/commands/` - comandos executaveis do fluxo.

`PROJECT.md` continua sendo a fonte unica da verdade do projeto Mi6.

Se houver divergencia entre este arquivo e os artefatos acima, prevalecem `PROJECT.md`, `LOGUS.md`, `CLAUDE.md`, `dialogos/` e `.claude/commands/`.
